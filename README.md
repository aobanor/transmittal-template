# transmittal-template

A professional Excel transmittal sheet for formally issuing documents between companies and project teams.

## Fields Included
- Transmittal Number and Date
- From / To / Attention / Email
- Purpose / Action Required (dropdown options)
- Document table (Doc No., Title, Rev, Copies, Format, Remarks)
- Remarks section
- Signature block (Issued By/ Received By)
